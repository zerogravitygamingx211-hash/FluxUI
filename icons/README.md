# FluxUI Icons and Graphics

## 🎨 Icon Collection

This directory contains all icons, logos, and graphics for FluxUI.

### **Application Icons**
- **fluxui.ico** - Main application icon (Windows)
- **fluxui.svg** - Scalable vector logo
- **fluxui_ascii.txt** - ASCII art logo

### **Icon Sizes**
- **fluxui_icon_16x16.png** - Small icon (16x16)
- **fluxui_icon_32x32.png** - Standard icon (32x32)
- **fluxui_icon_48x48.png** - Medium icon (48x48)
- **fluxui_icon_64x64.png** - Large icon (64x64)
- **fluxui_icon_128x128.png** - Extra large icon (128x128)
- **fluxui_icon_256x256.png** - HD icon (256x256)

### **Desktop Entry**
- **fluxui-ide.desktop** - Linux desktop entry file

### **Icon Information**
- **fluxui_icon_info.txt** - Icon metadata and information
- **fluxui_icons.json** - Icon configuration

## 🎯 Usage

### **Windows Applications**
```flux
# Use .ico files for Windows apps
ICON: "icons/fluxui.ico"
```

### **Web Applications**
```html
<!-- Use SVG for scalable graphics -->
<img src="icons/fluxui.svg" alt="FluxUI">
```

### **Cross-Platform**
```flux
# Use PNG files for compatibility
ICON: "icons/fluxui_icon_256x256.png"
```

## 📋 Icon Specifications

- **Format:** ICO, SVG, PNG
- **Sizes:** 16x16 to 256x256 pixels
- **Colors:** Modern blue theme
