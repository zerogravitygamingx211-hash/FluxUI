# FluxUI Scripts

## 📁 Python Source Files

This directory contains all Python source code for FluxUI.

### **Core Language Files**
- **fluxui.py** - Main language interpreter
- **parser.py** - Language parser
- **tokenizer.py** - Tokenizer
- **engine.py** - Execution engine
- **ui_engine.py** - UI rendering engine
- **components.py** - UI components library
- **parser_ast.py** - AST parser
- **renderer.py** - UI renderer
- **flux_runner.py** - Program runner

### **CLI and IDE**
- **fluxui_cli.py** - Command-line interface
- **fluxui_ide.py** - Integrated Development Environment

### **Installers and Tools**
- **fluxui_wizard_compact.py** - Setup wizard
- **fluxui_auto_installer.py** - Auto installer
- **fluxui_online_installer.py** - Online installer
- **build_all_executables.py** - Build system
- **organize_github_files.py** - Repository organizer

### **Utilities**
- **create_icon.py** - Icon generation
- **generate_pdf.py** - PDF documentation
- **global_install.py** - Global installation
- **simple_icon.py** - Simple icon creator

### **Testing**
- **_test_parser.py** - Parser tests
- **_validate_theme.py** - Theme validation

## 🔧 Usage

### **Running Directly**
```bash
python Scripts\fluxui.py program.flux
python Scripts\fluxui_ide.py
```

### **Building Executables**
```bash
python Scripts\build_all_executables.py
```

### **Installation**
```bash
python Scripts\fluxui_wizard_compact.py
```
