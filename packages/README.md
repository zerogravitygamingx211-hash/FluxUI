# FluxUI Platform Packages

## 📦 Download Packages

This directory contains platform-specific installation packages for FluxUI.

### **🖥️ Windows Package**
- **File:** `FluxUI_MasterWindows.zip`
- **Size:** ~137MB
- **Platform:** Windows 7/8/10/11
- **Architecture:** x64

### **🐧 Linux Package**
- **File:** `FluxUI_MasterLinux.zip`
- **Size:** ~180KB (source only)
- **Platform:** Ubuntu, CentOS, Arch Linux, etc.
- **Architecture:** x64

## 🚀 Quick Download

### **Windows Users**
1. Download `FluxUI_MasterWindows.zip`
2. Extract to folder
3. Run `FluxUI_Installer.exe` as administrator
4. Follow installation steps

### **Linux Users**
1. Download `FluxUI_MasterLinux.zip`
2. Extract: `tar -xzf FluxUI_MasterLinux.zip`
3. Run: `sudo ./install_global.sh`
4. Follow installation steps

## 🔧 Installation Requirements

### **Windows Requirements:**
- Windows 7/8/10/11
- Administrator privileges
- 200MB disk space
- .NET Framework (included)

### **Linux Requirements:**
- Python 3.8+
- tkinter package
- sudo privileges
- 100MB disk space
- X11/Wayland display
