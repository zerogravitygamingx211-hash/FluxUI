# 🚀 FluxUI Complete Setup Package

## 📦 Installation Options

### 🎯 Option 1: Double-Click Auto Installer (Easiest)
**File:** `Install_FluxUI.bat`

1. **Double-click** `Install_FluxUI.bat`
2. **Wait** for automatic installation
3. **Done!** FluxUI is fully installed

---

### 🧙‍♂️ Option 2: Interactive Wizard
**File:** `setup_fluxui.bat`

1. **Run** `setup_fluxui.bat`
2. **Follow** the 6-step wizard
3. **Choose** installation options
4. **Complete** installation

---

### ⚙️ Option 3: Manual Installation
**File:** `install_global.bat`

1. **Right-click** and "Run as administrator"
2. **Manual** installation process

## 🎮 After Installation

### 💻 Command Line Usage
```bash
# Check installation
fluxui --version

# Run programs
fluxui program.flux

# Modern CLI
fluxui-cli new myapp
fluxui-cli run program.flux

# Launch IDE
fluxui-ide
```

### 🖱️ File Operations
- **Double-click** .flux files → Execute with FluxUI.exe
- **Right-click** .flux files → More options
- **Drag & drop** .flux files onto FluxUI.exe

### 🎨 IDE Features
- **Syntax highlighting** for FluxUI
- **Project management** with file explorer
- **Integrated console** for output
- **Code templates** and snippets
- **Build and run** tools
