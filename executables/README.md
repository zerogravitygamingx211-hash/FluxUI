# FluxUI Executables

## 📦 Available Downloads

### **FluxUI v1.0.0 Installer**
- **File:** `FluxUI_v1.0.0_Installer.bat`
- **Type:** Complete installer with all components
- **Size:** Small download, installs everything
- **Usage:** Right-click → "Run as administrator"

### **Core Executables**
- **FluxUI.exe** - Main language interpreter
- **fluxui-cli.exe** - Command-line tools
- **fluxui.exe** - Alias for main interpreter

### **Installation Instructions**

1. **Download:** Get `FluxUI_v1.0.0_Installer.bat`
2. **Run as Admin:** Right-click → "Run as administrator"
3. **Follow Prompts:** Press any key to continue
4. **Complete:** Installation finishes automatically

### **What Gets Installed**

- **Program Files:** `C:\Program Files\FluxUI\`
- **System PATH:** Global command-line access
- **File Associations:** .flux files open with FluxUI
- **Desktop Shortcut:** Easy access to FluxUI
- **Documentation:** Complete language reference
- **Sample Files:** Example programs

### **After Installation**

Test with:
```bash
fluxui --ver
```

Should output: `FluxUI Beta 1.0`

### **System Requirements**

- **Windows:** 7, 8, 10, 11
- **Admin Rights:** Required for installation
- **Disk Space:** ~50MB
- **Architecture:** 64-bit recommended
