# FluxIDE Release

## Overview
FluxIDE is a complete Integrated Development Environment for the FluxUI programming language. This standalone executable provides everything you need to create, edit, and run FluxUI applications.

## Features
- **Syntax Highlighting** - Full color-coded syntax highlighting for FluxUI language
- **Code Editor** - Professional code editor with line numbers and auto-indent
- **File Explorer** - Project file navigation and management
- **Project Wizard** - Step-by-step project creation with templates
- **Code Templates** - Pre-built templates for common UI patterns
- **Integrated Runner** - Run and debug FluxUI code directly from the IDE
- **Console Output** - Real-time output and error display
- **Build Tools** - Create standalone executables from your FluxUI projects

## Installation
1. Download `FluxIDE.zip` from this folder
2. Extract the zip file to your desired location
3. Double-click `FluxIDE.exe` to run (no installation required)

## System Requirements
- Windows 10 or later
- No additional dependencies required

## Usage
1. **Launch** - Double-click `FluxIDE.exe`
2. **New Project** - Use File → New Project or the Project Wizard
3. **Open File** - File → Open to load existing `.flux` files
4. **Run Code** - Press Run button or F5 to execute your code
5. **Build EXE** - Tools → Build Executable to create standalone apps

## File Associations
- `.flux` - FluxUI source code files

## Quick Start
```
1. Start FluxIDE
2. Create a new project using the wizard
3. Select "Basic Application" template
4. Write your FluxUI code
5. Click Run to see your application
```
