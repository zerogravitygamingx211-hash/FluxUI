# FluxUI Shell Scripts

## 📁 Batch and Shell Scripts

This directory contains all installation and utility scripts for FluxUI.

### **Windows Batch Files (.bat)**
- **FluxUI_Installer.bat** - Simple installer
- **FluxUI_v1.0.0_Installer.bat** - Versioned installer
- **Install_FluxUI.bat** - Installation script
- **Install_FluxUI_Online.bat** - Online installer
- **build_release.bat** - Build release
- **install_flux_extension.bat** - Extension installer
- **install_global.bat** - Global installation
- **Run_Admin_Installer.bat** - Admin installer launcher
- **setup_fluxui.bat** - Setup script

### **Unix Shell Scripts (.sh)**
- **install_global.sh** - Unix global installation

## 🔧 Usage

### **Windows**
```cmd
# Run installer
Sh\FluxUI_Installer.bat

# Build release
Sh\build_release.bat

# Global install
Sh\install_global.bat
```

### **Unix/Linux**
```bash
# Global install
./Sh/install_global.sh
```

## 📋 Script Functions

- **Installation** - Install FluxUI to system
- **Building** - Compile and create executables
- **Setup** - Configure system integration
- **Utilities** - Helper scripts and tools

## 🎯 Requirements

- **Windows:** Administrator privileges for installation
- **Unix/Linux:** sudo privileges for global installation
- **Python:** Required for some scripts
